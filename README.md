# FipiranFunds

A Python library for fetching and exporting Iranian mutual fund data from Fipiran API.

## Features

- Fetch mutual fund data from Fipiran API
- Support for Jalali (Persian) date input
- Export data to CSV files
- Concurrent data fetching for multiple dates
- Automatic retry mechanism for failed requests
- Filter by specific fund registration numbers

```bash
pip install fipiranfunds
